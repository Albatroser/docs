# Instabot

Полезные скрипты для раскрутки аккаунтов Instagram.

## Что это?

Instabot - это модуль для языка Python, в котором не только реализована обертка над Instagram API, но и разные полезные функции, такие как "подписаться на список людей", "пролайкать фотки по хэштегам", "отписаться от невзаимных подписчиков" и тд. Instabot достаточно умный: [почитайте](Filtration.md), например, как он фильтрует людей, на которых собирается подписаться.

## Обновление

Так как проект Instabot молодой и активно развивается, то обновления будут выходить довольно часто. Поэтому если вы наткнулись на ошибку, не спешите бить в колокола: попробуйте обновить Instabot - может быть, эту ошибку уже поправили.

``` python
pip install -U instabot
```
